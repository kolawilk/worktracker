# UI: Kategorie-Farben als Border-Color

**Status:** 📋 Backlog  
**Priority:** P2  
**Parent:** Feature 002

---

## Problem
Die gewählte Farbe wird aktuell nur als kleiner Balken angezeigt. Schöner wäre die Border-Color der Kachel.

## Lösung
- Gewählte Farbe = Border-Color der Kachel
- Statt kleiner Farb-Balken unten
- Konsistent mit dem Design

## Akzeptanzkriterien
- [ ] Farbe beeinflusst Border-Color
- [ ] Kein kleiner Balken mehr
- [ ] iPad-optimiert
- [ ] Keine TypeScript Fehler
