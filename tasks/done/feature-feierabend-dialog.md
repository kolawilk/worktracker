# Feature: Feierabend-Dialog mit Statistik

**Status:** 📋 Backlog  
**Priority:** P2 (Nice-to-have)  
**Parent:** Feature 004

---

## Beschreibung
Wenn Arbeitstag beendet wird, erscheint ein Dialog mit:
- Netter Feierabend-Spruch (zufällig aus Auswahl)
- Ring-Diagramm (Tortendiagramm als Ring) mit Zeitverteilung
- Gesamtarbeitszeit

## Zusätzlich
- Dialog lässt sich über Button im Menüband aufrufen (dann ohne Spruch)

## Akzeptanzkriterien
- [ ] Dialog erscheint bei Arbeitszeit-Ende
- [ ] Zufälliger humorvoller Spruch
- [ ] Ring-Diagramm zeigt Zeitverteilung
- [ ] Aufrufbar über Menüband-Button
- [ ] iPad-optimiert
