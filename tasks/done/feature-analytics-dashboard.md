# Feature: Auswertungs-Dashboard (Grafana-like)

**Status:** 📋 Backlog  
**Priority:** P2  
**Parent:** Feature 005

---

## Beschreibung
Dashboard für detaillierte Auswertungen — inspiriert von Grafana, aber simpler.

## Anforderungen

### Zeitbereiche
- Spezifische Zeiträume auswählbar (von-bis)
- Vordefinierte: Heute, Diese Woche, Dieser Monat, Letzte Woche, Letzter Monat

### Auswertungen
- **Pro Wochentag:** Durchschnittliche Arbeitszeit pro Tag (Mo, Di, Mi...)
- **Monatsebene:** Monatliche Übersicht
- **Wochenebene:** Wochenübersicht
- **Trends:** Zeitverlauf über Zeitraum

### Filter
- Nach Kategorien filtern
- Mehrere Kategorien auswählbar

### Visualisierung
- Charts/Graphs (einfach, nicht überladen)
- Zeitverteilung als Pie/Donut
- Trend als Line-Chart
- Tabellarische Übersicht

## Akzeptanzkriterien
- [ ] Zeitbereiche auswählbar
- [ ] Wochentags-Auswertung
- [ ] Monats-Auswertung
- [ ] Wochen-Auswertung
- [ ] Trend-Anzeige
- [ ] Kategorien-Filter
- [ ] iPad-optimiert
- [ ] Keine TypeScript Fehler
