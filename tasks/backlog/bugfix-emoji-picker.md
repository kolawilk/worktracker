# Bugfix: Emoji Picker in Kategorie-Dialog

**Status:** 📋 Backlog  
**Priority:** P1  
**Parent:** Feature 002

---

## Problem
Im "Neue Kategorie" Dialog ist ein Fragezeichen Emoji (❓) als Platzhalter. Nutzer erwarten einen funktionierenden Emoji Picker.

## Lösung
- Entweder: Emoji Picker Library integrieren (z.B. `emoji-picker-react`)
- Oder: Einfache Emoji-Auswahl mit häufigen Emojis als Grid
- Oder: Native Emoji-Input mit besserer UX

## Akzeptanzkriterien
- [ ] Emoji Picker ist funktional
- [ ] Häufige Emojis schnell auswählbar
- [ ] Kein Fragezeichen als Platzhalter mehr
- [ ] Mobile/iPad optimiert
