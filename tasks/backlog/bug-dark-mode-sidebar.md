# Bug: Dark Mode wirkt sich nicht auf Sidebar aus

**Status:** 📋 Backlog  
**Priority:** P2  
**Type:** Bug

---

## Problem
Im Dark Mode bleibt die linke Menüleiste (Sidebar) hell statt dunkel zu werden.

## Erwartetes Verhalten
Sidebar sollte im Dark Mode die dunkle Theme-Farbe annehmen.

## Tatsächliches Verhalten
Sidebar bleibt hell/weiß.

## Betroffene Komponenten
- Linke Navigation/Sidebar
- Vermutlich: `src/components/ui/` oder Navigation-Component

## Akzeptanzkriterien
- [ ] Sidebar passt sich Dark Mode an
- [ ] Icons/Text in Sidebar sind im Dark Mode lesbar
- [ ] Keine visuellen Glitches beim Theme-Wechsel
