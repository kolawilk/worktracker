# UI: Emoji-Animation abschalten

**Status:** 📋 Backlog  
**Priority:** P1  
**Parent:** Feature 003

---

## Problem
Die Animation des Emojis in der aktiven Kachel ist zu aufdringlich und lenkt ab. Das Dashboard steht den ganzen Tag auf dem iPad und sollte nicht ablenken.

## Lösung
- **Pulse-Animation komplett entfernen**
- Stattdessen: Statische Hervorhebung (z.B. farbiger Border, Schatten)
- Oder: Einfache Opacity-Änderung

## Akzeptanzkriterien
- [ ] Keine Animation mehr
- [ ] Aktiver Status trotzdem klar erkennbar
- [ ] Keine Ablenkung mehr
